## 🔍 Search and Match Guidelines

### Contract Similarity Assessment

When starting a new contract review, assess similarity with historical cases along these dimensions:

#### 1. Contract Type (合同类型)
- **Exact Match**: Same contract type (e.g., lease agreement)
- **Related Type**: Similar contract type (e.g., lease vs. license agreement)
- **Different Type**: Different category

#### 2. Parties Involved (当事人)
- **Same Counterparty**: Reviewed contracts with same counterparty
- **Similar Counterparty**: Reviewed contracts with similar parties
- **Different Counterparty**: New counterparty

#### 3. Transaction Background (交易背景)
- **Same Project**: Related to same project or transaction
- **Similar Context**: Similar business context or purpose
- **Different Context**: Different business context

#### 4. Key Issues (关键问题)
- **Similar Risk Profile**: Similar types of risks identified
- **Similar Legal Issues**: Similar legal or compliance issues
- **Different Issues**: New or different types of issues

### Similarity Scoring

Assign similarity score based on matches:
- **High Similarity (80-100%)**: Contract type + counterparty match
- **Medium Similarity (50-79%)**: Contract type match only
- **Low Similarity (20-49%)**: Partial overlap in issues
- **No Similarity (0-19%)**: Different contract types and contexts

## 🎯 Key Learnings from Historical Cases

### Lease Agreements (租赁合同)

#### Recurring Issues
1. **Security Deposit (押金)**
   - Frequency: ⭐⭐⭐⭐⭐ (Always appears)
   - Standard Issue: 2-3 months rent amount
   - Key Risk: Unclear return timeline and conditions
   - Standard Solution: Specify return within 15 days after lease termination

2. **Termination Rights (解除权)**
   - Frequency: ⭐⭐⭐⭐ (Very common)
   - Key Risk: Landlord unilateral termination without cause
   - Standard Solution: Require mutual termination rights with reasonable notice

3. **Maintenance Responsibility (维修责任)**
   - Frequency: ⭐⭐⭐ (Common)
   - Key Risk: Unclear allocation of maintenance obligations
   - Standard Solution: Detailed breakdown of responsibilities

#### Best Practices
- Always review from both landlord and tenant perspectives
- Specify detailed acceptance criteria for premises condition
- Include clear rent escalation clauses (if applicable)
- Define sublease and assignment rights clearly

### Construction Contracts (建设工程合同)

#### Recurring Issues
1. **Payment Milestones (付款节点)**
   - Frequency: ⭐⭐⭐⭐⭐ (Always appears)
   - Key Risk: Milestones not linked to actual progress
   - Standard Solution: Link payments to measurable completion percentages

2. **Change Orders (变更订单)**
   - Frequency: ⭐⭐⭐⭐ (Very common)
   - Key Risk: No clear procedure for scope changes
   - Standard Solution: Detailed change order procedures with cost and timeline adjustments

3. **Delay Penalties (延期违约金)**
   - Frequency: ⭐⭐⭐ (Common)
   - Key Risk: Excessive daily penalties or unclear calculation
   - Standard Solution: Cap penalties at reasonable percentage of contract value

#### Best Practices
- Verify construction qualifications and permits
- Include detailed scope of work and technical specifications
- Specify acceptance criteria and testing procedures
- Include warranties and defect liability period

### Framework Agreements (框架协议)

#### Recurring Issues
1. **IP Ownership (知识产权归属)**
   - Frequency: ⭐⭐⭐⭐⭐ (Critical for tech collaborations)
   - Key Risk: Unclear ownership of background IP and foreground IP
   - Standard Solution: Explicitly define IP ownership for all categories

2. **Fee Structure (费用结构)**
   - Frequency: ⭐⭐⭐⭐ (Very common)
   - Key Risk: Vague or unclear fee arrangements
   - Standard Solution: Detailed fee schedule with payment timing

3. **Termination Clauses (终止条款)**
   - Frequency: ⭐⭐⭐ (Common)
   - Key Risk: Unfair termination rights or penalties
   - Standard Solution: Balanced termination rights for all parties

#### Best Practices
- Clearly define project scope and deliverables
- Specify payment milestones tied to deliverables
- Include confidentiality obligations
- Address force majeure and unforeseen circumstances

### Purchase Contracts (采购合同)

#### Recurring Issues
1. **Vague Acceptance Criteria (验收标准模糊)**
   - Frequency: ⭐⭐⭐⭐⭐ (Always appears in purchase contracts)
   - Key Risk: Acceptance standards not specific enough, leading to disputes
   - Standard Solution: Detailed step-by-step acceptance criteria with technical parameters

2. **Unclear Breach Liability (违约责任不明确)**
   - Frequency: ⭐⭐⭐⭐⭐ (Critical risk)
   - Key Risk: No specific penalty amounts or calculation methods
   - Standard Solution: Define penalties for late delivery, quality issues, and non-performance

3. **Dispute Resolution Location (争议解决地)**
   - Frequency: ⭐⭐⭐⭐ (Very common)
   - Key Risk: Dispute resolution in supplier's location, unfavorable to buyer
   - Standard Solution: Specify buyer's location courts for dispute resolution

4. **Payment Terms (付款条款)**
   - Frequency: ⭐⭐⭐ (Common)
   - Key Risk: Payment periods too long or unclear conditions
   - Standard Solution: Clear payment terms (e.g., 30 days after acceptance + invoice)

#### Best Practices
- Always attach detailed technical specifications for equipment purchases
- Include quality assurance period with response time requirements
- Specify warranty terms and repair/replacement procedures
- Define penalty structures for different types of breaches
- Include acceptance test procedures with pass/fail criteria
- Clarify delivery terms (Incoterms if international)
